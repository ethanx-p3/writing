# Analytics Reports

This directory contains per-article marketing and performance reports generated by the Marketing Agent.

## File naming

`<slug>_marketing.md` — distribution copy and performance tracking

## Performance Review Cadence

- **7-day check**: Fill in initial performance metrics (opens, clicks, LinkedIn impressions)
- **30-day check**: Final performance assessment, subscriber attribution
- **Quarterly**: Cross-article analysis — which topics, formats, and hooks perform best

## Cross-Article Analytics

Periodically, the pipeline runner should produce a cross-article summary:
- Top performing articles by opens, clicks, new subscribers
- Topic patterns that resonate
- Best-performing hook formats
- Optimal publish day/time based on open rates

This feeds back into article prioritization in `pipeline/queue.json`.
