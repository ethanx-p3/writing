# Ideas Inbox

The daily agent appends new article ideas here each morning. Review, edit, and promote the ones you want.

**To promote an idea:**
1. Copy the idea block below to a new file at `articles/_ideas/<slug>.md` (expand it using `templates/idea.md`)
2. Add an entry to `pipeline/queue.json` with `"stage": "idea"`
3. Delete or archive the idea from this file

**To discard an idea:** just delete its block from this file.

**To iterate on an idea:** edit it directly here. The agent reads this file each morning before generating new ideas, so it will see your edits and avoid duplicating them.
